
========================
Leonardo Bootstrap Admin
========================

Compatibility layer beetween ``django-admin-bootstrap`` and Leonardo CMS / FeinCMS

.. contents::
    :local:

Installation
------------

.. code-block:: bash

    pip install leonardo-bootstrap-admin

Disable admin
-------------

.. code-block:: bash

    pip uninstall leonardo-bootstrap-admin
